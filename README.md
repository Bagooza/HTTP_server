# Tiny Python HTTP Server

A minimal Python HTTP server that serves HTML pages using raw sockets.  
This project is designed to help you learn how HTTP requests, responses, and web servers work from scratch â€” without any frameworks.

---

## Features

- Serves HTML pages from a local `htdocs` folder
- Uses raw Python sockets (`socket` module)
- Portable: works on any machine with Python 3
- Includes a sample styled `index.html` page
- Simple and easy to extend for multiple pages
